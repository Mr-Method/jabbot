= Mac 上的軟體包裝與散佈

== Drag'n'Drop -- DMG

* DMG = Disk Image . 可掛戴、可燒錄。
* Application Bundle
** 應用程式 = 目錄
** 執行檔
** 資料檔
** 函式庫
** Install = 丟到「應用程式」資料夾
** Uninstall = 丟到垃圾桶
* 以 DMG 散佈 = Application

== Installer -- PackageMaker

* XCode 的工具之一
* 製做標準的「安裝程式」所用
* 用處
** 所安裝的不是應用程式（如 Kernel Extension、Library、輸入法）
** 要將檔案安裝到多個目錄時
* MetaPackage
** 大包裝
* 特色
** 有圖形製作套件的界面
** 有命令列界面，便於自動處理

== Darwinports

* 也是 jkh@ 的傑作
* 獨立的介面 : "port"
** port install mutt
** port uninstall mutt
** port upgrade mutt
** port outdated
* 新的概念: activate / deactivate
** port deactivate apache 1.34
** port activate apache 2.0.55
* 做好各種套件
** port pkg mutt
** port dmg mutt
** port rpmpackage mutt
* 特色
** 不用自已生 plist
** 可安裝多個版本，且容易「切換」（activate/deactivate）
** 已經裝過的軟體，會自動作一份套件給日後備用
** 獨立運作，可跨不同平台使用
